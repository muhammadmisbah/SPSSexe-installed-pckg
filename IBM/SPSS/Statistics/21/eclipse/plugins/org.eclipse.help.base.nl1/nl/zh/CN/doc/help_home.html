使用 Eclipse 帮助系统

在左边的目录框架（）中浏览主题。单击某个主题以使其显示。使用后退和前进按钮来浏览已查看的主题的历史记录。

搜索

要在文档中快速找到有关特定内容的主题，请在搜索字段中输入查询。使用搜索框架（）来显示“搜索”视图。可以通过只选择感兴趣的部分来缩小搜索范围。

同步

单击在目录中显示按钮（）就会在导航树中选择该主题。与内容链接按钮（）使导航树与当前主题同步。

功能

要显示有关应用程序中已禁用的功能的文档，可选择显示所有主题按钮（）。这将影响目录和搜索结果。

关键字索引

要显示关键字的按字母顺序排序的索引，请选择索引帧（）。并非所有产品都包含索引。

更多信息

如果在联机帮助中找不到问题的答案，请访问我们的 Web 站点（网址为 eclipse.org）以查找相关的文章并加入 Eclipse 社区。
