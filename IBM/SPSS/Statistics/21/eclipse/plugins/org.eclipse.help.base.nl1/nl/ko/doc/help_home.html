Eclipse 도움말 시스템 사용

왼쪽의 컨텐츠 프레임()에 있는 항목을 찾아보십시오. 표시할 항목을 누르십시오. 표시된 항목의 히스토리 내에서 탐색하려면 이전 및 다음 단추를 사용하십시오.

검색

문서에서 특정 주제에 대한 항목을 신속하게 찾으려면 검색 필드에 조회할 내용을 입력하십시오. 검색 보기를 표시하려면 검색 프레임 ()을 사용하십시오. 관심있는 섹션만 선택하여 검색의 범위를 좁힐 수 있습니다.

동기화

목차에서 표시 단추()를 클릭하면 탐색 트리에서 해당 항목이 선택됩니다. 컨텐츠가 포함된 링크 단추()를 사용하여 탐색 트리의 선택사항을 현재 항목에 연결할 수 있습니다.

기능

애플리케이션에서 사용 불가능한 기능에 대한 문서를 표시하려면, 모든 주제 표시 단추()를 선택하십시오. 목차 및 검색 결과에 적용됩니다.

키워드 색인

알파벳순으로 정렬된 키워드 색인을 표시하려면 색인 프레임 ()을 선택하십시오. 모든 제품이 색인에 포함되지 않습니다.

자세한 정보

온라인 도움말에서 질문에 대한 응답을 찾을 수 없는 경우, 웹 사이트 eclipse.org에서 관련 주제를 찾아보거나 Eclipse 커뮤니티에서 확인할 수 있습니다.
