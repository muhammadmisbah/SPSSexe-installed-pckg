שימוש במערכת העזרה של Eclipse

אתרו נושאים במסגרת התוכן () מימין. לחצו על נושא כדי להציג אותו. השתמשו בלחצן אחורה ובלחצן קדימה כדי לנווט בהיסטוריה של הנושאים שהוצגו.

חיפוש

כדי לאתר במהירות נושאים מסוימים בתיעוד, הקלידו שאילתה בשדה חיפוש. השתמשו במסגרת החיפוש () כדי להציג את תצוגת החיפוש. תוכלו לבחור רק את החלקים שאתם מעוניינים בהם כדי לצמצם את היקף החיפוש.

סינכרון

לחיצה על הלחצן הצגה בתוכן העניינים () תבחר את הנושא בעץ הניווט. הלחצן קישור לתוכן () שומר על הסינכרון בין עץ הניווט לבין הנושא הנוכחי.

יכולות

כדי להציג תיעוד על היכולות המושבתות ביישום, בחרו את הלחצן הצגת כל הנושאים (). פעולה זו משפיעה על תוכן העניינים ועל תוצאות החיפוש.

אינדקס מילות מפתח
