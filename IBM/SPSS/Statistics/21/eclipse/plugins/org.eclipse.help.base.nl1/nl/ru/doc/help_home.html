Работа со справочной системой Eclipse

Просмотрите список разделов, приведенный во фрейме Оглавление () слева. Для открытия раздела щелкните на нем. Кнопки Назад и Вперед позволяют перемещаться по хронологии просмотренных разделов.

Поиск

Для быстрого поиска в документации разделов, посвященных конкретному предмету, введите искомый термин в поле Поиск. Фрейм Поиск (() позволяет открыть панель Поиск. Выбрав только интересующие вас разделы, вы можете сузить область поиска.

Синхронизация

Нажмите кнопку Показать в оглавлении () для выделения раздела в дереве навигации. Кнопка Связать с оглавлением () позволяет синхронизировать дерево навигации с текущим разделом.

Группы функций

Для просмотра документации о группах функций, выключенных в приложении, нажмите кнопку Показать все разделы () . Это приведет к изменению оглавления и результатов поиска.

Индекс ключевых слов

Для просмотра упорядоченного по алфавиту индекса ключевых слов откройте фрейм индекса (). Индекс предусмотрен не во всех продуктах.

Дополнительная информация

Если вам не удалось найти ответ на вопрос в электронной справке, обратитесь к Web-сайту eclipse.org, на котором вы можете найти дополнительные материалы, а также принять участие в сообществе Eclipse.
