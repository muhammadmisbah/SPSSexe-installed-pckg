Utilisation du système d'aide Eclipse

Parcourez les rubriques dans le cadre Table des matières () sur la gauche. Cliquez sur une rubrique pour l'afficher. Utilisez les boutons Précédent et Suivant pour naviguer dans l'historique des rubriques consultées.

Recherche

Pour trouver rapidement les rubriques d'aide sur un sujet particulier dans la documentation, entrez une requête dans la zone Rechercher. Utilisez le cadre Recherche () pour afficher la vue Recherche. Vous pouvez affiner la portée de votre recherche en ne sélectionnant que les sections qui vous intéressent.

Synchronisation

Cliquer sur le bouton Afficher dans la table des matières () entraîne la sélection de cette rubrique dans l'arborescence de navigation. Le bouton Lier au contenu () permet de synchroniser l'arborescence de navigation à la rubrique en cours.

Capacités

Pour afficher la documentation sur les capacités qui sont désactivées dans l'application, sélectionnez le bouton Afficher toutes les rubriques (() . Cette opération affecte la table des matières et les résultats de la recherche.

Index des mots clés

Pour afficher un index par ordre alphabétique des mots clés, sélectionnez le cadre Index (). Tous les produits n'ont pas forcément d'index.

Pour en savoir plus

Si vous ne trouvez pas de réponse à votre question dans l'aide en ligne, visitez le site eclipse.org pour rechercher des articles et prendre part à la communauté Eclipse.
