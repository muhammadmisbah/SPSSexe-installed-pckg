使用 Eclipse 說明系統

請瀏覽左邊內容頁框 () 中的主題。 按一下某個主題，會使它顯示出來。 請利用上一頁和下一頁按鈕來導覽已檢視的主題歷程。

搜尋

若要在文件中迅速找到特殊主旨的相關主題，請在搜尋欄位中輸入查詢。 請使用搜尋頁框 () 來顯示「搜尋」視圖。您可以只選取您有興趣的章節來縮小搜尋的範圍。

同步化

按一下顯示在目錄中按鈕 （） 時， 會在導覽樹狀結構中選取該主題。鏈結內容按鈕 （） 可讓導覽樹狀結構維持與現行主題同步。

功能

若要顯示應用程式中停用的功能相關文件， 請選取顯示所有主題按鈕 ()。 這會影響目錄和搜尋結果。

關鍵字索引

若要顯示按字母排序的關鍵字索引，請選取索引頁框 （）。並非所有產品都包含索引。

其他資訊

如果在線上說明中找不到問題的答案，請造訪我們的網站 eclipse.org 來尋找文章及參與 Eclipse 社群。
